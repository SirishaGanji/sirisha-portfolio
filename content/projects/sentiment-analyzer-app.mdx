## Overview
A lightweight **Flask** web app that analyzes free text and CSV uploads with **VADER**. It summarizes **positive / neutral / negative** proportions and renders a pie chart.

## Features
- Paste text or upload a CSV (`text` column)  
- Batch analysis with Pandas + VADER  
- Pie chart output (Matplotlib)  
- Download annotated CSV

## Tech Notes
- VADER for rule-based sentiment  
- Robust CSV parsing w/ Pandas  
- Simple HTML templates + Bootstrap (or Tailwind)
