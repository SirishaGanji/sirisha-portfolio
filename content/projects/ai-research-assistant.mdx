## Overview
An agent-based **AI assistant** that performs multi-step research: search → retrieve → rank → summarize → draft a brief with citations.

## Capabilities
- Query expansion & multi-source retrieval  
- Chunk ranking and deduplication  
- Structured summary with inline citations  
- Exports to Markdown / PDF

## Architecture
- **FastAPI** service  
- **LangChain** tools for search & RAG  
- Caching to reduce token usage
